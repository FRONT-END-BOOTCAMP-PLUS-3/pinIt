## 📌 주요 기능

1. **포토스팟 등록**

   - 사용자가 자신의 포토스팟을 사진과 함께 등록 가능
   - 위치 기반으로 포토존 검색 가능
   - 지도 위에 포토스팟을 띄워줌

2. **‘좋아요’ 리스트**

   - 마음에 드는 핀에 '좋아요'를 누르고 저장
   - 좋아요한 핀 모아보기 제공

3. **포토 챌린지**

   - 매주 제공되는 챌린지 주제에 따라 핀 등록 가능
   - ‘좋아요’ 순으로 인기 핀 노출
   - 챌린지 결과 리스트 제공

4. **관리자 기능**
   - 게시글 및 회원 관리
   - 챌린지 주제 및 기간 설정

## ⚙️ 기술 스택

- **프론트엔드**: Next.js, Typescript, SCSS, CSS Modules
- **백엔드 및 데이터베이스**: Supabase
- **기타**: PWA
